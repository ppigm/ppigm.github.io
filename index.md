# Welcome to PPIGM's official Python Learning Platform! 🇮🇩
Here you'll learn Python basics. 
#### Contributing
If you see anything worth changing, you can always implement that change yourself! Consult the `markdown_101.md` for a quick introduction to Markdown.
1. Have `node` installed.
1. Fork the repository, and clone it to your local machine. 
1. Install the required components. Run the following:
```sh
npm install gitbook-cli
nvm use 12.18.1
```
1. Familiarize yourself with the development environment. It's pretty simple.
1. Using a code or text editor of your choice (I recommend Visual Studio Code), make changes in the appropriate markdown (`.md`) files, corresponding to each part of the units. 
1. Alternatively, add a new `.md` file, and link it appropriately to `SUMMARY.md` to make it appear on the table of contents in the left.
1. Once you're happy with your creation, make a pull request to the repository.
